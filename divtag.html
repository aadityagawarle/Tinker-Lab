Java

Java is a high-level, object-oriented programming language designed to let programmers write once, run anywhere

It is widely used for developing applications ranging from mobile apps to enterprise software, running on billions of devices worldwide.

Python

Python is a high-level, interpreted programming language known for its clear syntax and strong support for multiple programming paradigms.

It is widely used for web development, data analysis, artificial intelligence, and automation, making it extremely popular among both beginners and professionals.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation and design of documents written in HTML or XML.

It enables developers to control layout, colors, fonts, and responsiveness, allowing for visually rich and consistent web pages across multiple devices.
